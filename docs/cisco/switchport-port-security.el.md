switchport port-security
========================

Η εντολή `switchport port-security` ενεργοποιεί την ασφάλεια στη συγκεκριμένη 
διεπαφή. Χρειάζεται να ακολουθήσουν εντολές που παραμετροποιούν την ασφάλεια.
